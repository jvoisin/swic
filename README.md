# Simple Web Interface for Calibre

As written on the tin, a simple web interface for your [calibre](https://calibre-ebook.com) database.

## Features

- Search on various fields
- Download the files
- Written in a non-awful language
- Minimal amount of dependencies
- Trivial deployment: drop the static binary somewhere, run it with `swic
  -library /path/to/your/calibre/library`.
- Read-only access to the filesystem
- Responsive interface
- No javascript
- No user management, slap a basic auth in front of it.
- Minimal amount of (ugly) code as it was written in an evening.
